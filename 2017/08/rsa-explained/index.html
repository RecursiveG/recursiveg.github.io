RSA加密简述

本来去年学完基础数论就想写的，结果不知不觉又拖了一年……

本文中使用到的符号：

P: 明文 (Plaintext)
C: 密文 (Ciphertext)
E: 加密函数 (Encrypt)
D: 解密函数 (Decrypt)
K: 密钥 (Key)
𝜑: 欧拉函数

对称加密

信息加密是人类生活中重要的一部分，凯撒密码大概是普通人最熟悉的一种密码了。加密时，它将原字母用字母表中固定偏移量后的字母替代。解密就是反过来替换：

E("ABC",3) => "DEF"
D("DEF",3) => "ABC"

简单易懂。

对称加密存在的一个问题是，为了能够进行加密通信，双方必须持有相同的密钥。
但如果密钥用明文发送的话，一旦被截取，那么之后的通信就不再安全了。
于是一个天才的想法出现了：
既然对称加密的问题在于，用来加密的密钥也可以用来解密，那么我们能不能设计一种方法，
使得用来加密的密钥不能用来解密呢？

非对称加密

RSA就是这样一种“用来加密的密钥不能用来解密”的算法。具体算法如下：

  1. 生成两个素数$p$和$q$
  2. 计算他们的乘积$n=pq$
  3. 计算欧拉函数$\varphi(n)=\varphi(pq)=(p-1)(q-1)$
  4. 寻找一个数$k$使其和$\varphi(n)$互质，即$gcd(k,\varphi(n))=1$
  5. 寻找$k$模$\varphi(n)$的模反元素$j$，即$k j\equiv1(mod\ \varphi(n))$

我们可以验证

$$M^{kj}\equiv M(mod\ n)$$

因为

$$M^{kj}=M^{c\varphi(n)+1}=M(M^{\varphi(n)})^c$$
